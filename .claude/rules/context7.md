# Consulta de Documentação via Context7

**IMPORTANTE:** Quando não tiver certeza sobre API, sintaxe ou padrões de uso de qualquer biblioteca do stack, **SEMPRE** consulte a documentação atualizada via Context7 antes de implementar.

## Quando usar

- Incerteza sobre assinatura de função, hook ou padrão de uso
- Verificar se um padrão não está deprecado
- Implementando feature com biblioteca que pode ter mudanças recentes
- Debugging de comportamento inesperado que pode ser uso incorreto da API

## Workflow

1. Resolver o ID da biblioteca: `mcp__plugin_context7_context7__resolve-library-id`
2. Consultar a documentação: `mcp__plugin_context7_context7__query-docs`
